# Fig S3.1: Analyses of correlation between enhancer and promoter activity

This directory contains R and Perl scripts to reproduce plots shown in Figure S3.1 of Supplementary
Note S3. Scripts need to be run sequentially to reproduce all analyses. To run scripts, bedtools, R
and Perl need to be installed, as well as the following R packages:

```
data.table
scam
ggplot2
showtext
ks
```

All required input data can be optained by downloading the `input` directory from synapse and
placing it into this directory: https://www.synapse.org/Synapse:syn74843290.
