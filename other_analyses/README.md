# Additional analyses
Code in this directory was used to perform following additional analysis:

- Code and reference files in the `enhancer_synergy` directory were used to produce the analysis for
the section 'Super-additive functions of distal enhancers,' including Fig. 5 and Extended Data Fig. 
9
- The functions in `chromatin_categorization.R` were used to define and annotate element-gene pairs
with chromatin categories
- `promoter_class_analysis.R` was used to produce the analysis for Supplementary Note 4, including
Supplementary Fig. N4.1
- Code in `3d_contact_analyses` was used to perform analyses related to 3D contact shown in Fig. 4
and Supplementary Figure N2.1 (Supplementary Note 2)
- `extended_data_fig6_eg_properties` contains code used to generate locus plots in Extended Data
Fig. 6 and the "few vs. many enhancers" analysis in Extended Data Fig. 6e
- `supplementary_fig5_eg_properties` contains code used to compute properties across different E-G
predictors shown in Supplementary Figure 5
- `supplementary_figN3.1_ep_correlation` contains code used to perform the enhancer-promoter
correlation analyses presented in Supplementary FigN3.1 (Supplementary Note 3).
- `cartoons_fig1a` contains helper scripts used to simulate and plot Hi-C and eQTL data for
illustrations in Fig. 1a.
